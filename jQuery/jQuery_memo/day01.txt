jQuery 문법
1) closest() : 부모를 찾기
2) find() : 자식에서 찾기
3) prev() : 이전 요소 가지고 오기
4) next() : 다음 요소 가지고 오기
5) eq(인덱스) : 해당 요소를 찾기
6) attr("키", "값") : 해당 속성의 값을 변경
7) remove() : 해당 요소를 삭제
8) append("<태그>내용</태그>") : 해당 요소를 추가
9) html() = innerHTML
10) on() = addEventListener()
11) .css() : css의 속성값을 변경하고 싶은 경우
12) .prop() : js의 프로퍼티를 변경하고 싶은 경우
13) .each() = forEach()
14) .text() : 값을 출력하거나 변경하고 싶은 경우
