테이블(table)
  : 여러 종류의 데이터가 행과 열로 정리된 표 
  
  tr : 행을 의미
  th, td : 열을 의미


이미지 삽입 방법 => 절대 경로 혹은 상대 경로 활용
    / : 최상위 폴더의 경로
    ./ : 현재 폴더의 경로
    ../ : 현재 폴더의 위에 있는 폴더의 경로

이미지 태그 사용 방법
<img src="" alt="대체 이미지" />
	1) 최상위 폴더 경로를 이용한 이미지 경로
	<img src="/day02/images/icon01.png" alt="대체 이미지" />

	2) 현재 폴더의 경로를 이용한 이미지 경로
	<img src="./images/icon02.png" alt="대체 이미지">

	3) 현재 폴더의 상위 폴더를 이용한 이미지 경로
	<img src="../day02/images/icon03.png" alt="대체 이미지">

a 태그 사용
: 페이지 내에서 북마크 기능으로 사용 가능
	_self : 현재 페이지(기본값)
	_blank : 새 탭
	_parent : 부모 페이지로 이동(iframe 이용했을 때 사용)
	_top : 최상위 페이지로 이동(iframe 이용했을 때 사용)
