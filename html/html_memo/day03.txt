시멘틱 태그(Semaintic tag)
- 의미가 있는 태그들로, HTML 문서 내의 가독성 개선과 외부 검색 엔진(SEO)에도 내 문서 내의 어떤 부분이 메인인지 보여줄 수 있도록 최적화 시켜줄 수 있는 부분을 담당
- 개발자의 유지 보수 및 웹 페이지를 시각적이 아니라, 음성으로 읽어주는 '스크립리더'를 이용하거나 웹 접근성에 유용

시멘틱 태그의 종류
- header : 상단, 헤더를 의미
- nav : 메뉴, 내비게이션을 의미
- section : 여러 중심 내용을 감싸는 공간을 의미
- main : 내용의 중심을 의미
- article : 글자가 많이 들어간 부분을 의미
- aside : 사이드에 위치하는 공간을 의미
- footer : 하단, 푸터를 의미

form 태그
- 폼은 사용자가 웹 사이트에 데이터를 전송해주기도 하며 이 밖에 웹 페이지가 입력 데이터를 사용하기 위해서도 사용 가능

1. 속성(Attribute)이란?
	- 태그에 추가로 정보를 제공하거나 태그의 동작이나 표현을 제어할 수 있도록 지정해주는 설정값

2. 속성의 사용 방법
	- 속성명="속성값"

3. 표기법(Case)
	- 카멜 표기법 : 낙타의 등을 본따 만든 표기법(camelCase)
		- Java, JavaScript 언어에서 주로 사용

	- 케밥 표기법 : HTML에서는 케밥 표기법 사용(kebab-case)
		- HTML에서 사용

	- 파스칼 표기법 : 낙타 등이 양봉으로 되어있는 표기법(PascalCase)
		- Java에서는 클래스명, 함수명을 표기할 때 사용
		- Python에서는 클래스명을 표기할 때 사용

	- 스네이크 표기법 : 뱀이 기어가는 모양을 본따 만든 표기법(snake_case)
		- Python에서 주로 사용

input 태그
- 사용자로부터 데이터를 입력 받을 수 있는 태그

input 태그의 name과 value
- form 태그 내에서 input 속성의 이름, 값을 찾기 위한 목적으로 사용
- value 입력 태그의 초기 값

input 태그 type 종류
1. text : 문자열의 형태로 사용자에게 데이터를 입력 받음
2. password : text와 동일하지만 입력 시에 데이터가 보이지 않음
3. datalist : 사용자가 input 요소에 데이터를 입력할 때 미리 정의된 옵션을 드롭 리스트로 보여줌으로써 자동 완성 기능 제공
4. textarea : input text 외의 텍스트를 사용할 수 있는 태그
5. radio : 단일 선택할 시에 사용
6. label : 대신 클릭할 수 있게 도와주는 태그
7. checkbox : 다중 선택 시 사용하는 속성
8. select : 다중 값들 중 선택할 수 있는 다른 태그
9. button 태그 - submit, reset, button
10. color
11. email
12. url
13. datetime-local, date
14. number
15. file
16. range

CSS란?
- CSS는 Casecading Style Sheets의 약자
- CSS는 HTML 요소들이 각종 미디어에서 어떻게 보이는 가를 정의하는 데 사용
- 스타일을 HTML 문서로부터 분리하는 것이 가능

CSS를 사용하는 이유?
- HTML만으로 웹 페이지를 제작하는 경우, HTML 요소의 세부 스타일을 일일이 따로 지정해주어야 하기 때문에 많은 시간 소요
- 완성 후에도 스타일의 변경 및 유지 보수가 어려움
- 이 문제점을 해결하기 위해 사용

CSS 문법
p { textalign: center; color: blue; }
[선택자] { 속성명: 속성값; 속성명: 속성값; }

CSS 선택자
1. 전체 선택자
	- 스타일을 모든 요소에 적용할 때 사용
	- 주로 모든 하위 요소에 한꺼번에 스타일을 적용할 때 사용
	- 전체 선택자는 asterrisk(*) 기호 사용

2. 태그 선택자
	- 특정 태그가 쓰인 모든 요소에 스타일 적용
	- 선택자 위치에 태그명을 작성하면 사용된 모든 해당 태그에 동일한 스타일 적용

3. 클래스 선택자(class 속성에 작성된 값)
	- 클래스 선택자는 특정 집단의 여러 요소를 한 번에 선택할 때 사용
	- 같은 클래스의 이름을 가지는 요소들을 모두 선택해주고 스타일 적용 시 선택자에 ".클래스명" 작성

4. 아이디 선택자(id 속성에 작성된 값)
	- 아이디 선택자는 특정 요소를 선택할 때 사용
	- 스타일을 지정할 때 사용되며, 선택자 부분에 "#아이디명" 작성

5. 그룹 선택자
	- 여러 선택자에 같은 스타일을 적용할 경우 쉼표로 구분해서 여러 선택자를 나열한 후 스타일은 한 번만 정의
