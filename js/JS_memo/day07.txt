Event
- 권장 사용 형식 : addEventListener("이벤트명", 콜백 함수)

자주 사용되는 이벤트
1) click : 요소를 클릭한 경우

2) mouseover : 요소에 마우스를 올린 경우

3) mouseleave : 요소에서 마우스가 떠난 경우

4) mousedown : 요소를 누르고 마우스를 아직 때기 이전 상태

5) mouseup : 요소를 누르고 마우스를 땐 상태, 마우스 위치가 요소 위에 있어야 함

6) mousemove : 마우스가 움직일 때마다

7) mousewheal : 마우스 휠이 이동된 경우

8) scroll : 스크롤의 변화가 생긴 경우

9) focus : 활성화 되어 있는 경우

10) blur : 포커스에서 벗어난 경우

11) keypress : 키를 누른 순간

12) keyup : 키를 누르고 있다가 땐 순간

13) keydown : 키를 누르고 때기 이전인 순간

14) change : 폼 필드의 속성이 변경된 경우
