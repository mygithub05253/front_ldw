useRef란?
- 저장 공간 또는 DOM 요소에 접근하기 위해 사용되는 React의 훅함수
- 여기서 Ref는 Reference의 뜻으로 참조한다는 의미

불변성이란?
- 원시데이터(string, number, bollean, null, undefined, symbol) : 불변성이 존재
- 원시데이터는 수정 시, 메모리에 저장된 값 자체는 변경할 수 없고, 새로운 메모리 저장 공간에 새로운 값을 저장
- 원시 데이터가 아닌 데이터(Object 타입 : 객체, 배열, 함수, ... )는 불변성이 없음
- 원시 데이터가 아닌 데이터는 수정했을 때 기존에 저장되어 있던 메모리 저장 공간의 값 자체를 변경

불변성이 중요한 이유
=> React가 화면을 리랜더링 하는 기준
- state가 변하면 리랜더링
- state가 변하지 않으면 리랜더링이 되지 않음

불변성 유지
- 바벨 es5로 바꿔주는 문법
- 모던 자바스크립트가 편하자고 만든 문법의 종류
- 실제 배포 시 모던 자바스크립트 문법이 변경되어야 함
- 그것을 도와주는 것이 바벨
- 바벨을 통해 es6 -> ex5로 다운 그레이드하여 배포
- 하지만 배열의 많은 메서드들은 es5에서도 사용 가능
