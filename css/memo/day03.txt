flex 속성
1) flex-grow : 남은 공간을 얼마만큼 차지할 것인지와 관련된 속성

2) flex-shrink : 축소 요소를 설정하는 속성
- 선행 조건 : height => flex 항목의 축소 요소를 비율에 맞게 축소시키는 설정

3) flex-basis : flex 항목의 박스 아이템의 브라우저 width에 따른 비율을 정의할 때 사용

Position 
1) static : 일반적인 문서의 흐름에 따라 배치한다.(default)
2) relative : 각 요소들을 하나로 겹치게 한 뒤 z-index를 사용해야 할 때 사용한다.
3) absolute :  부모의 relative 요소를 찾은 뒤 해당 테두리 영역 내에서 움직이도록 한다.
4) fixed : 문서상의 흐름을 무시하고, 최상위 조상 요소를 기준으로 잡는다.
5) sticky : 문서의 스크롤을 따라오게 만든다.
	- 단, 부모 요소에 overflow 속성이 있다면 사용이 불가능하다.
